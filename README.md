# Godot_PlatformerCharacterController

A simple character controller for platformer games in Godot

## 📥 Installation
### Linux / MacOS
Copy and paste this command into your terminal at the project root:
```bash
clone-gd-addon https://github.com/ChillCube/Godot_PlatformerCharacterController.git
```

Full documentation can be found in [DOCUMENTATION.md](./DOCUMENTATION.md)
